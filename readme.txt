This plugin implements a "like/dislike" rating system for comments

== Description ==

This plugin implements a "like/dislike" rating system for comments. In the setting page you can choose a "compact / splitted" mode that show ratings like sum or separately. 

== Installation ==

1. Unzip and upload the plugin in your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag & Drop the widget in your sidebar and customize your options
